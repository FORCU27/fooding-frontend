# ceo-front 전체 QA 태스크

## 실행 준비
- [ ] Node 18+, Yarn 4 환경 확인 후 루트에서 `yarn install` 완료
- [ ] `.env.sample` 기준으로 환경변수 정리 및 `apps/ceo-front/.env` 동기화
- [ ] 개발 서버 `yarn dev:ceo`, 빌드 검증 `yarn build --filter=ceo-front`, 타입/린트/테스트 `yarn type-check --filter=ceo-front && yarn lint --filter=ceo-front && yarn test --filter=ceo-front`
- [ ] 테스트 계정/스토어/쿠폰 등 데이터 확보, 계정 전환 시 로컬 스토리지/쿠키 초기화
- [ ] Chrome 데스크톱/모바일 뷰 우선, 가능하면 Safari 한 번 교차 확인

## 공통 동작/가드
- [ ] 로그인/회원가입/비밀번호 오류/세션 만료 시 처리, 로그아웃 후 스토어 선택 상태 초기화 확인
- [ ] `/select-store` 스토어 선택: 목록/빈 상태/검색 또는 정렬 존재 시 동작, 선택 후 전역 상태 반영, 권한 없는 스토어 접근 시 가드
- [ ] 헤더/내비 라우팅: 활성 탭 표시, 새로고침/딥링크 시 상태 유지 여부
- [ ] 로딩/에러 핸들링: Spinner/Toast 노출, 재시도 버튼 또는 재요청 동작

## 홈/공지/뉴스
- [ ] 루트 `/`: 로그인 여부에 따른 CTA, 최근 공지/이벤트 로딩/빈/에러 상태
- [ ] `/notice` 정적 공지: 내비게이션 링크, 더보기 CTA 동작, 브라우저 히스토리 처리

## 마이 영역 핵심 플로우
### 스토어 관리 (`/my/store/*`)
- [ ] 기본정보/부가정보/시설: 필수값 검증, 저장 성공/실패 토스트, 새로고침 후 반영
- [ ] 좌석안내: 좌석 추가/삭제/정렬, 안내 문구 수정, 비어 있는 경우 안내 표시
- [ ] 사진: 업로드/삭제/대표 설정, 용량·확장자 제약, 업로드 실패/중단 시 UX
- [ ] 메뉴: 카테고리 생성/수정/삭제/순서 변경, 메뉴 생성/수정/삭제, 대표·추천 배지, 이미지·메뉴판 업로드, 카테고리 전환 시 캐시/로딩, 스토어 미선택 시 가드

### 리워드/쿠폰 (`/my/reward/*`)
- [ ] 포인트 적립 내역: 정렬 토글, 페이지네이션, 전화번호 마스킹, 빈 상태/에러 표시
- [ ] 쿠폰 리스트: 상태 토글(발급/일시중단), 정렬·페이지네이션, 삭제/취소 다이얼로그, 상세 페이지 이동
- [ ] 포인트샵·쿠폰 발급/수정: 유효기간·수량 검증, 시간대 처리, 중복 발급/만료 시 메시지

### 단골/리뷰 (`/my/regular/*`)
- [ ] 단골/즐겨찾기/소식 구독: 목록 로딩, 필터·정렬 존재 시 동작, 빈 상태
- [ ] 리뷰: 목록 표시, 답글 작성/수정/삭제, 본인 확인, 리뷰 없음 상태 메시지, 업로드/전송 실패 토스트

### 알림/디바이스/통계
- [ ] 알림 목록: 읽음/미확인 상태, 필터 또는 일괄 읽음, 알림 클릭 시 링크 이동
- [ ] 디바이스: 목록/상세 `/my/devices/[id]`, 등록·연결/해지 흐름, 상태 뱃지, 권한 없는 디바이스 접근 처리
- [ ] 통계 `/my/statistics`: 날짜 선택 시 데이터 갱신, 카드 값·증감률 포맷, 차트 렌더링, 데이터 없음/에러 시 안내

### 기타 컨텐츠
- [ ] `my/news`/`notice` 등 컨텐츠 페이지 링크·외부 이동 시 새 창 여부, 빈 상태/로딩 처리
- [ ] 로그아웃 후 히스토리/스토어 캐시 초기화 및 재로그인 시 정상 복원

## 단말/브라우저/비기능
- [ ] 반응형: 375px 모바일~데스크톱에서 헤더/표/버튼 깨짐 여부, 호버 의존 기능의 터치 대응
- [ ] 접근성: 주요 버튼/링크 aria-label, 키보드 포커스 이동, 토스트·다이얼로그 포커스 트랩
- [ ] 성능/안정성: 초기 로딩 지연 시 스켈레톤 노출, 이미지 지연 로딩, 콘솔·네트워크 에러 여부
- [ ] 지역화/시간대: 날짜·시간·통화 포맷(ko-KR) 일관성, 로컬 타임존 기준 확인
